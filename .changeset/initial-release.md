Initial release: headless interaction hooks (`useImageZoomPan`,
`useSlideNavigation`, `useGestureHandler`, `useBarMeasure`, `useBodyScrollLock`,
`useFocusTrap`) plus a batteries-included `<ImageViewer>` shell with zoom/pan/pinch/
swipe, render slots (header, header actions, footer, overlay), `ViewerContext`
coordination (`setContentShift`, measured bar heights), CSS-variable theming, inline
default icons, focus trap, and reduced-motion support.
